## v1.8.9 변경사항

### 1. 예약발행 99개 한도 정지 버그 수정
- 예약 목록 가상 스크롤로 실제보다 적게 세던 문제 → 네이버 "예약 발행 N건" 라벨 기준으로 정확히 판정. 99건에서 정지.

### 2. 이용기간 만료 차단 시스템
- 만료 시 크롤링 현황 봇1에 "이용기간 만료" 표시(2pt 크게, 빨강).
- 크롤링 시작 / 포스트 글쓰기 / 생성된 포스트 보기·업로드 / 업로드 시 차단 + 안내.
- 명의별(아이디 드롭다운 활성 계정) 만료 판정. 명의 전환 시 봇1 즉시 갱신.
- 빈값 = 사용불가(차단). 무제한은 "무제한" 버튼(2099-12-31)으로 부여. 관리자 만료일 편집기에 "사용불가" 버튼 추가.
- 신규가입: 1명의 30일 무료. 관리자는 항상 예외.

### 3. 지역 설정 정리
- 세종 = "세종시" 단일(구단위). 시도명이 구 이름에 겹치면 중복 제거(세종시/제주시).

### 4. 저장한 프롬프트가 글쓰기에 적용되도록 수정
- 편집기 저장 경로(APPDATA)와 글 생성 읽기 경로 불일치로 저장분이 무시되던 버그 수정 → 경로 통일.
- AI 자동생성은 현재 기본 프롬프트 기반으로 업종별 프롬프트 창작.
