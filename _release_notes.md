## v1.8.2 변경사항

### 네이버 플레이스 크롤러 Throttle(차단) 우회 강화
- **자동 봇 조절**: 크롤 대상 구(區)가 10개 이상이면 동시 봇을 자동으로 2봇으로 하향해 후반부 throttle(수집 5개 잘림)을 회피. 구가 적으면 설정대로 유지해 속도 우선.
- **스트래글러 자동 복구**: 전체 크롤 종료 후, 차단으로 '딱 5개'에서 멈춘 구만 자동 리스트업해 순차로 1회 더 재시도 → 누락 구 자동 복구.
- 서울 25구 기준: 기존 3봇 7구 누락 → 자동 2봇 + 복구로 무손실 수집 확인.

### 봇 운용 정책 2봇으로 고정
- 봇 개수 옵션을 1·2로 정리(3봇 제거), 기본값 2봇으로 변경.
- 크롤링 현황 패널도 2칸으로 정리.
- RAM 기반 자동 상한도 최대 2봇으로 조정(4GB 미만 PC는 1봇).
