### 1. 키워드 크롤링 업종 필터 제거 (`places_crawler.py`)
- **문제**: "부산개인회생" 등 키워드 검색 시 업체 0개 수집
- **원인**: `_match_biz` 업종 필터가 "법무사" 카테고리 업체를 무관 업종으로 판단해 전부 스킵

- **결과**: 네이버 지도에 나오는 검색 결과 전부 수집, 2차 필터에서 거름

### 2. F8 포스팅 선택 다이얼로그 트리 구조 변경 (`main.py`)
- **변경 전**: 날짜 콤보박스 + 2단계(그룹→업체) 트리
- **변경 후**: 3단계(날짜→그룹→업체) 트리, 날짜별 접힘(초기 collapsed)
- 날짜 헤더: bold, 파란색(#1e40af), 연파랑 배경(#eff6ff), "📅 MM-DD (N개)" 형식
- 전체선택/해제도 3단계 순회로 업데이트
- 선택 개수 상단에 표시

### 3. 일반 사용자 API 키 저장 버그 수정 (`settings_dialog.py`)
- **문제**: 비관리자가 API 키 입력 후 저장해도 저장 안 됨
- **원인**: `_save()` 비관리자 분기에서 API 키를 cfg에서 제거 후 저장
- **수정**: 비관리자도 `_persist_api_keys_for_user` 플래그 세팅해서 로컬 저장 + Firebase 동기화
- **결과**: 다른 컴퓨터에서 로그인해도 API 키 유지
